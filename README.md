# Weather Forecast Analysis Dashboard

## Overview
An interactive weather dashboard built using Power BI to analyze and visualize weather conditions.

## Features
- Current Temperature
- Humidity Analysis
- Wind Speed Monitoring
- Pressure Tracking
- UV Index
- PM10 Air Quality
- Chance of Rain Forecast
- Sunrise & Sunset Information  
- Multi-City Filtering

## Tools Used
- Power BI
- Power Query
- DAX
